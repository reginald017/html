Listas

Listas são o conjunto de actividades a serem executadas, detalhadas com um padrão, em HTML, temos 2 tipos de listas esxistentes.E elas são: OL-Lista Ordenada, e UL-Lista não Ordenada.

Elementos em uma lista

Os elementos em uma lista são colocados com a tag li(List Item).Ex: <li>elemento</li>.

Listas Ordenadas(<OL>)

As listas ordenadas são aquelas listas em que os seus itens possuem uma ordem ou numeração. Uma lista é representada com a tag ol(ordered list).Ex:<ol></ol>

  1. item 1
  2. item 2
  3. item 3
  4. item 4

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parâmetro são:

  • 1 - Valor padrão. Cria listas numeradas. Ex: 1, 2, 3, 4, …
  • A - Cria listas alfabéticas em maiúsculas. Ex: A, B, C, D, …
  • a - Cria listas alfabéticas em minúsculas. Ex: a, b, c, d, …
  • I - Cria listas com algarismos romanos em maiúsculas. Ex: I, II, III, IV, …
  • i - Cria listas com algarismos romanos em minúsculas. Ex: i, ii, iii, iv, …

Podemos também escolher onde a nossa lista vai começar com o parâmetro start. Ex: <ol type= "I" start=5>
A lista vai ser exibida assim: V, VI, VII, VIII, IX, …

Listas Não Ordenadas (<ul>)

Listas não Ordenadas são aquelas listas que não seguem uma determinada ordem, e elas são representadas com a tag <ul>

  • Item 1
  • Item 2
  • Item 3

O marcador padrão é a bolinha preta totalmente preenchida (circle), mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

  • disc - padrão. Uma bola preta totalmente pintada
  • circle - Uma bola com uma borda preta e sem preenchimento
  • square - Um pequeno quadrado preto totalmente pintado

Listas Mistas

Podemos também criar listas mistas, configurando listas dentro de outras listas. Veja o exemplo a seguir:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
    4. Item 2.4
    5. Item 2.5
  3. Item 3
  4. Item 4
  5. Item 5

Note que os itens da lista interna (a, b, c, …) está deslocado para a direita em relação ao item 2, do qual essa sub-lista é filha.

OBS: Além de aninhar listas ordenadas, podemos juntar listas <ul> com <ol> e vice-versa. As listas internas sempre terão deslocamento interno para a direita.

Lista de Definições

São aquelas listas em que os seus elementos serão descritos, ela é representada pela tag <dl> e cada termo que se pretende descrever é representado pela tag <dt> e cada descrição é representada pela tag <dd>.
