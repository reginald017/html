Principais Formatações

Negrito/Destaque

Para colocar um texto em negrito, utilizamos a tag <b>texto</b> - bold; (Não Semântica).

Nesta frase, tenho um termo em destaque, usando a tag <strong>texto<strong> - forte;(Semântica).

Itálico/Enfâse

Nesta frase, tenho um termo itálico usando a tag <i>texto<i>(não semântica).

Nesta frase, temos um termo em enfâse usando a tag EM, <em>texto</em> (Semântica).

Marcado

Nesta frase, tenho um termo marcado, usando a tag <mark>texto</mark>

Texto grande e pequeno

O <big>big</big> morreu, mas o SMALL AINDA SOBREVIVE, para diminuir o texto, usamos a tag <small>texto</small>.

Texto Deletado

Nesta frase, temos um termo deletado, para deletar um texto usamos a tag <del>texto</del>.

Texto inserido

Nesta frase, temos um termo inserido, que é o oposto do texto deletado, e esta tag anteriormente era usada como underline(u), para inserir um texto, usamos a tag <ins>texto<ins>

Textos sobrescritos e textos subscritos

Para sobrescrever um texto, como este X2 usamos a tag <sup>2</sup>, e para subscrever(sub-índice), como este X2, usamos a tag <sub>2</sub>

Trechos de código

Nós somos (ou seremos) programadores, e por isso compartilhamos muito código fonte em diversas linguagens. Para isso, existe a tag <code> da HTML onde você pode delimitar seu código. A principal vantagem no uso dessa tag é a o valor semântico que ela representa, indicando ao navegador que se trata de um código de computador. Porém, existe também um efeito visual, pois as letras ficam no modo mono-espaçadas (monospace), o que facilita bastante a leitura do código. Analise, por exemplo, o trecho de código a seguir:

Código Python

num = int(input("Digite um número")); if(num%2==0) printf("Número Par"); else: printf("Número ímpar"); printf("Fim do Programa");

Você pode achar que o código vai aparecer lindamente no seu site, porém o que acontece é simplesmente uma mudança das letras, para que elas fiquem mono-espaçadas (veja a imagem).

Para resolver esse problema, vamos usar uma outra tag HTML chamada <pre>, que: mantém o texto pré-formatado, exatamente da mesma maneira na qual ele foi digitado, incluindo quebras de linhas, espaços e tabulações. 


            num = int(input("Digite um número"));

            else:

            printf("Fim do Programa");

"Citações"

Para adicionar uma citação em HTML, usamos a tag <q>, que vem da palavra em inglês quote, que significa citação, essa tag é usada dentro de parágrafos.Ex: Só sei que nada sei-Aristóles

podemos adicionar a palavra cite dentro da tag para explicar a fonte da citação, para efectuarmos uma citação fora de um parágrafo, usamos a tag <blockquote cite="https://www.reginaldo.com">citação</blockquote>. Ex:
